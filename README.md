# Minecraft Clients

A collection of popular Minecraft clients commonly used on servers like 2b2t and other anarchy/survival servers.

> ⚠️ Disclaimer  
> I am NOT responsible for anything that happens from downloading or using these clients.  
> Some unofficial or cracked clients may contain malware, RATs, or other harmful software.  
> Always download from official sources and scan files before running them.

> ⚠️ Intended Use  
> These clients should only be used on anarchy servers such as 2b2t or servers where hacked clients are allowed.  
> Using them on regular multiplayer servers may break server rules and can result in bans.

# Safety Tips

- Prefer open-source clients whenever possible.
- Download only from official websites or GitHub pages.
- Never trust random MediaFire or Discord downloads.
- Scan files with VirusTotal before opening them.
- Use an alt account when testing clients.
- Keep important passwords and accounts secured.
- Avoid cracked clients unless you fully trust the source.
